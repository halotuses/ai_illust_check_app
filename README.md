## Overview
イラストをアップロードすると、そのイラストがAIによって作成された可能性を表示します。

## Requirement
-macOS Ventura ver13.5.1  
-Docker version 24.0.5  


## Usage
１）jpg,pngのイラストを”イラストを選択してください”ボタンからアップロードする。  
２）CHECKをクリック。  
３）loading後にRESULTに結果が表示されます。
